## 五、Qlib vs 聚宽对比

| 维度 | 聚宽 | Qlib |
|:--|:--|:--|
| 部署 | 云端 SaaS | 本地 Conda |
| 数据 | 平台 API | 自采 Baostock/AKShare |
| 策略范式 | 事件驱动规则策略 | 因子 + ML + 信号 |
| 因子 | 手写（股息率、PE 等） | Alpha158 等因子库 |
| 评价 | 收益/回撤 | 收益/回撤 + IC/ICIR |
| 适合 | 快速验证想法 | 因子研究、ML 实验 |

**结论：** 聚宽适合初学者快速验证规则策略；Qlib 适合系统性因子研究与机器学习实验，两者互补。
